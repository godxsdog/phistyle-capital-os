# Frontend

Next.js skeleton for the PhiStyle OS shell.

## Current Scope

- Provides a minimal first screen.
- Reads backend health from `NEXT_PUBLIC_API_URL`.
- Does not implement app registry, platform features, AI, investment logic, or legacy app integration.

## Local Run

```sh
cd frontend
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```
